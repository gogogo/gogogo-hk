The Gogogo Project aims to provide a comprehensive and easy-to-use transit planning service for Hong Kong. We want Gogogo to be the service you can rely on whenever you want to find out how to get from point A to point B within Hong Kong.

The Gogogo Project is currently building the service and maintaining the transit database with team members collaborating using open source methods. If you are interested join us! You may participate in our gogogo-hk-discuss google group. Apart from software developers, we will also need contributors for our the community-maintained transit database.

{% trans "Transit Data" %}

We Gogogo developers believe in the spirit of openness and free flow of information. Right now we are busy perfecting the data format and data entry interface, so the data is expected to be relatively unstable and inaccurate, but we do intend to release the transit information database to the public domain when the data is sufficiently accurate and comprehensive. As a corollary, all user contributions must necessarily be public domain.
